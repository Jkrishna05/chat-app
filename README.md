A **real-time chat application** built using **React, Node.js, MongoDB, and Socket.io**.
Users can create accounts, chat instantly, share images, see online status, and manage their profiles.

The application provides a **modern messaging experience similar to popular chat platforms**, with real-time updates and responsive UI.

# 🚀 Features

## 🔐 Authentication

* Secure **User Signup & Login**
* **JWT-based authentication**
* Protected API routes
* Persistent login sessions

---

## 💬 Real-Time Messaging

* **Instant messaging** using Socket.io
* **Typing Indicator** – Shows when another user is typing
* **Read Receipts** – See when messages have been read
* **Message Timestamps**
* **Smooth real-time message updates**

---

## 🖼️ Media Sharing

* **Image Sharing in Chat**
* Users can send and receive images instantly within conversations

---

## 📅 Smart Chat Experience

* **Date Separators** for messages (Today, Yesterday, etc.)
* Organized chat history similar to messaging apps like WhatsApp
* Clean conversation layout

---

## 🟢 User Presence

* **Online / Offline Status**
* **Last Seen Status** showing when a user was last active
* Real-time presence updates

---

## 👤 User Profile

Users can customize their profiles:

* Upload or change **Profile Picture**
* Add or edit **Bio**

* Profile updates reflected instantly in chat

## 🎨 UI / UX

* **Responsive design** (Mobile, Tablet, Desktop)

# ⚙️ How It Works

1. The **React frontend** provides the chat interface and communicates with the backend APIs.
2. **Node.js + Express** handle authentication, user management, and message APIs.
3. **MongoDB** stores users, conversations, and messages.
4. **Socket.io** enables real-time communication between connected users.
5. Images are uploaded and shared instantly within chats.
